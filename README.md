# Copernicus Data Download 🌊

Automated Python script for downloading and processing Copernicus Marine datasets.

---

## 📌 Overview
This project provides a Python-based workflow to download oceanographic data from the Copernicus Marine Service.  
It supports spatial subsetting, variable selection, and year-by-year data extraction.

---

## ⚙️ Features
- Download Copernicus Marine datasets
- Spatial subsetting (latitude & longitude)
- Time-based filtering
- Year-wise data download
- NetCDF output format
